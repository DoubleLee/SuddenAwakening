\subsection{Detailed Description}
The \hyperlink{class_tile_map_layer}{Tile\-Map\-Layer} class represents a layer on the map. It can draw itself to the map. 

It's made up of the vector of entities which are on this layer.

This class shouldn't be used by any devs directly, only within the \hyperlink{class_tile_map}{Tile\-Map} class.
